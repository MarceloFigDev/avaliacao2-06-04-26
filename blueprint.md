# Maracatu Tech Landing Page Blueprint

## Overview

This document outlines the design and development plan for a simple, visually appealing landing page for a technology event called "Maracatu Tech". The page will be built using modern HTML, CSS, and JavaScript, following the guidelines for creating a clean, responsive, and accessible user experience.

## Design and Style

*   **Color Palette:**
    *   Pearly White: `#FDFEFE`
    *   Pearly Gray: `#EAECEE`
    *   Light Blue: `#A9CCE3`
    *   Dark Blue (for text and accents): `#2980B9`
*   **Typography:** A clean, modern sans-serif font will be used for readability.
*   **Layout:** A single-page layout with a clear header, a main content section, and a footer. The layout will be responsive and adapt to different screen sizes.
*   **Visual Elements:** Subtle gradients and box shadows will be used to create a sense of depth and visual interest.

## Features

*   **Header:** A prominent header with the event name, "Maracatu Tech".
*   **Event Details:** A section displaying the date and location of the event.
*   **Call to Action:** A "Register Now" button that will be the primary call to action.
*   **Footer:** A simple footer with copyright information.
*   **Interactivity:** The "Register Now" button will display a confirmation message when clicked.

## Implementation Plan

1.  **HTML (`index.html`):**
    *   Structure the page with semantic HTML5 tags (`<header>`, `<main>`, `<footer>`, `<section>`).
    *   Add the main heading for the event.
    *   Include placeholders for event details.
    *   Add a call-to-action button.
    *   Link the CSS and JavaScript files.

2.  **CSS (`style.css`):**
    *   Define the color palette using CSS variables.
    *   Style the header, main content, button, and footer.
    *   Use Flexbox for layout and alignment.
    *   Add media queries for responsiveness.

3.  **JavaScript (`main.js`):**
    *   Create a simple web component for a styled button to demonstrate modern JavaScript practices.
    *   Add an event listener to the "Register Now" button to show a confirmation alert.
